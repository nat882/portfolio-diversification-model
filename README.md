# Portfolio Diversification & Efficient Frontier Model

An Excel-based portfolio optimisation tool analysing 7 assets across equities, commodities, and fixed income. The model uses Monte Carlo simulation across 2,000 random portfolio combinations to plot the efficient frontier and identify the optimal risk-adjusted allocation using the Sharpe ratio.

## Key Findings

**1. Optimal Allocation**

The maximum Sharpe ratio portfolio achieves an expected annual return of 20.63% at a volatility of 15.98%, compared to the equal weight benchmark which delivers 11.70% return at 12.20% volatility. Optimisation improves the Sharpe ratio from 0.66 (equal weight) to 1.01 (max Sharpe), a 53% improvement in risk-adjusted return.

| Portfolio | Exp. Return | Volatility | Sharpe Ratio |
|-----------|-------------|------------|--------------|
| Max Sharpe Ratio | 20.63% | 15.98% | 1.01 |
| Min Volatility | 5.30% | 6.82% | 0.12 |
| Equal Weight Benchmark | 11.70% | 12.20% | 0.66 |

**2. Diversification Benefit**

The correlation matrix reveals GLD as the strongest diversifier in the portfolio, showing a -0.37 correlation with BP — the most significant negative equity correlation in the universe. AGG also provides defensive value with negative correlations against AAPL (-0.08), MSFT (-0.13), and GSK (-0.17). The minimum volatility portfolio achieves just 6.82% volatility compared to 12.20% for the equal weight benchmark, a reduction of 5.38 percentage points driven by the low-correlation bond and gold allocations.

**3. Risk / Return Trade-off**

The minimum volatility portfolio reduces annualised risk from 15.98% to 6.82%, a reduction of 9.16 percentage points, but at the cost of 15.33% in expected annual return and a Sharpe ratio collapse from 1.01 to 0.12. An investor prioritising capital preservation would accept the minimum volatility allocation; a growth-oriented investor seeking to maximise risk-adjusted returns would opt for the maximum Sharpe portfolio.

## Methodology

**Log Returns**

Monthly log returns are calculated as LN(current price / previous price). Log returns are used rather than simple percentage changes because they are additive over time and their distribution is closer to normal, which is a requirement of the statistical analysis underpinning the model.

**Annualisation**

Returns are annualised by multiplying monthly averages by 12. Volatility is annualised by multiplying monthly standard deviation by the square root of 12, following the square root of time rule (variance scales linearly with time; volatility scales by its square root).

**Covariance Matrix**

A 7x7 covariance matrix is derived from the correlation matrix and individual asset volatilities. This encodes the complete risk structure of the asset universe and is used directly in the portfolio variance calculation.

**Portfolio Variance**

Portfolio variance is calculated using matrix multiplication: w' × Σ × w, where w is the vector of portfolio weights and Σ is the covariance matrix. In Excel this is implemented using nested MMULT and TRANSPOSE functions. This correctly accounts for all pairwise correlations between assets, unlike a naive weighted average of individual variances.

**Monte Carlo Simulation**

2,000 random portfolio weight combinations are generated by normalising random numbers to sum to 1. Each combination produces a portfolio return, volatility, and Sharpe ratio. Plotting volatility against return for all 2,000 portfolios reveals the efficient frontier — the boundary of optimal portfolios where no other combination offers the same return at lower risk.

**Sharpe Ratio**

The Sharpe ratio is calculated as (Portfolio Return − Risk Free Rate) / Portfolio Volatility, using 4.5% as the risk-free rate (approximate UK base rate over the period). It measures return per unit of risk and is used to identify the optimal portfolio.

## Limitations

1. **Historical returns as a proxy for expected returns.** The model uses 2019 to 2024 price data to estimate future performance, which is not guaranteed to hold. BP's -2.30% annualised return reflects a specific post-COVID energy market environment and may not be representative going forward.

2. **Static covariance matrix.** The covariance structure is assumed constant over the full period. In practice, asset correlations rise during market stress events such as the 2020 COVID crash, reducing diversification benefit precisely when it is most needed.

3. **Unadjusted closing prices.** The model uses unadjusted closing prices rather than dividend-adjusted prices, which marginally understates true returns for high-dividend assets such as GSK and BP.

4. **Monte Carlo approximation.** The model samples 2,000 random portfolios rather than solving analytically for the exact efficient frontier. A more rigorous approach would use Excel Solver or a quadratic programming tool to identify precise optimal weights at each return level.
